## Notes

- **P-4.3 baseline** (aaf2667): delta compression only — no game-loop snapshots.
  Bots had no active game state to serialize, so Out ≈ 1kbps and Delta Eff = 99%
  reflected near-empty snapshot batches.
- **P-5.x current**: bots now receive real HeroState snapshots each tick.
  FOW (P-5.1) culls non-visible entities; LOD (P-5.4) gates tier 1/2 entities
  to 50Hz / 20Hz respectively.  Out kbps reflects actual game traffic.
- **Tick budget**: P-5.x adds Phase 0b (PriorityEvaluator per observer, O(N²))
  and the rewind-buffer RecordTick call.  Any increase here is the cost of those.
- **Delta Efficiency**: expected to drop from 99% as heroes now move and their
  positions change each tick; FOW and LOD partially offset this.
